# UFC Load Testing Framework

This project provides automated load, stress, and performance testing for UFC multidev/staging environments using k6.

Features:
- realistic user journey simulation
- staged load profiles
- performance thresholds
- JSON/CSV reporting
- scalable distributed execution
- Grafana integration

Target environment:
https://develop.pantheon-multidev.ufc.com/

WARNING:
This framework is intended only for approved staging/multidev environments. Do not execute high-load tests against production systems without authorization.
